포고 렌더러 17.12 패치로 17.05 이후 버전에 사용할 수 있습니다.
바뀐 점은 아래와 같습니다.

17.12 패치
  1. ALSA nonblock 출력 사용.
  2. MPD 출력과 믹서 변경.
  3. MPD와 Upmpdcli 터미널 출력 방지.
  4. MPD 11월 버퍼 패치 적용.
  5. 커널 스케줄러 일부 변경.


포고와 라즈베리 렌더러가 비슷하게 오다가, 17.10 버전부터 차이가 나기 시작했습니다.
최신 트렌드(?)에 맞게 포고 렌더러를 업데이트하려고 하였으나,
하드웨어 없이 패치 만으로는 한계가 있는 것 같습니다.
그럼에도 고맙게도 킹앤드류님께서 확인해 주셔서 17.12 패치 버전을 만들었습니다.
메모리 재생이나 ALSA 라이브러리 컴파일은 적용하지 못했으나,
ALSA와 MPD 출력, MPD 버퍼, 커널 스케줄러 등 가능한 부분은 적용하였습니다.

패치는 쉘 스크립트로 만들었습니다.
인터넷이 연결된 포고에 SSH로 연결한 다음, 아래 명령어를 차례로 입력하면 됩니다.
그리고 시스템을 다시 시작하면 됩니다.

# apt-get update
# apt-get -y install curl ca-certificates
# curl -L https://goo.gl/81JALG -o pogo_audio_renderer_17.12_patch
# chmod +x pogo_audio_renderer_17.12_patch
# ./pogo_audio_renderer_17.12_patch

* 소프트웨어 볼륨을 사용하는 방법은 기존 17.05 블로그에 있습니다.

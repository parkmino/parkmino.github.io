포고 렌더러 17.09 패치로 17.05 버전에 사용할 수 있습니다. 

원래 이미지를 만들려고 했으나 여의치 않아서 패치 파일로 대체합니다.

이미지를 옮기는 것보다 시간은 덜 걸릴 겁니다.


바뀐 점은 아래와 같습니다.


17.09 패치

  1. ALSA 출력 재 설정.

  2. MPD 우선권 삭제. (?)

  3. MPD 버퍼 상세 설정.

  4. MPD 믹서 사용 안 함.

  5. 커널 스케줄러 일부 조정.

  6. 유튜브 스크립트 추가. (ut2mpd+youtube-dl)

  7. KBS클래식FM/CBS음악FM/TBSFM 라디오 추가. (kr2mpd)


위의 1~5번은 지금까지와 마찬가지로 틈틈이 들으면서 순도를 높이고 자연스럽게 들리도록 수정한 것입니다.

6번과 7번은 각각 유튜브와 인터넷 라디오 관련 내용인데 얼마나 쓰일지는 모르겠습니다.

ut2mpd 명령어에 유튜브 주소를 붙이거나 kr2mpd에 국내 라디오를 붙이면 됩니다. ('kr2mpd 1fm' 은 KBS 클래식 FM)

'ut2mpd'와 'kr2mpd'를 실행하면 간단한 설명이 나옵니다.

참고로 ut2mpd 셸 스크립트는 youtube-dl( https://rg3.github.io/youtube-dl/ )를 이용합니다.


패치는 셸 스크립트로 만들었습니다.

인터넷이 연결된 포고에 SSH로 연결한 다음 아래 명령어를 차례로 입력하면 됩니다.

설치 후 'Reboot? (y/n)'에 'y'를 입력하면 시스템이 재시작됩니다.


# wget https://goo.gl/VgdHE4 -O pogo_audio_renderer_17.09_patch

# chmod +x pogo_audio_renderer_17.09_patch

# ./pogo_audio_renderer_17.09_patch


* 앱의 소프트웨어 볼륨을 사용하는 방법은 기존 17.05 블로그 내용에 추가하였습니다.

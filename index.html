알릭스 렌더러 18.04입니다.
알릭스 사용자 분께 소프트웨어를 설치해 드리고 백업 용으로 만든 이미지입니다.
Alix 3D2 모델에 8GB CF 기준입니다.

https://drive.google.com/file/d/1Z3IZEjhYm-Qp-VJuhQNLlmSj_bBoVfWh/view?usp=sharing

주요 사양은 아래와 같습니다.

-. Voyage Linux 11.0 Daily Build (Kernel 4.1.6, Debian Jessie 8.7)
-. ALSA 라이브러리 1.1.5
-. MPD 0.20.18
-. Upmpdcli 1.2.15
-. USB 자동 마운트
-. SSH (사용자:root, 비밀번호:voyage)
-. CIFS, NFS
-. 인터넷 라디오, kr2mpd(KBS/MBC/CBS/TBS/국악방송)
-. 기타 튜닝

1. Voyage Linux

제가 Real Time 실시간 커널을 좋아하지 않아서 일반 커널의 Voyage Linux를 선택했습니다.
데비안 Jessie 버전인데, 개발자가 최신 Stretch 지원은 계획이 없다고 합니다.
패키지를 업데이트하면 Voyage 저장소의 공개키 오류로 진행이 되지 않습니다.
따라서 Voyage 패키지는 사용하지 않고 데비안 패키지 만으로 설정했습니다.

2. USB 자동 마운트

포고 렌더러와 같이 USB 자동 마운트를 지원합니다.
MPD 클라이언트에서 DB를 업데이트하면 연결된 USB의 음원이 보일 겁니다.

3. CIFS와 NFS 지원

cifs-utils와 nfs-common 패키지를 설치하여 CIFS와 NFS를 지원합니다.
이로써 프로세스가 추가되기는 하는데, 기존 방식으로 사용하시는 분들도 있는 것 같아서요.
PC나 NAS 음원을 직접 마운트해서 사용하시려면 /etc/fstab 파일을 수정하시면 됩니다.

4. DSD 지원

기존 렌더러와 마찬가지로 DoP(DSD over PCM)로 기본 설정해 두었습니다.
Native DSD를 시험해 보시려면, Putty 등으로 SSH로 연결해서 아래 명령어를 실행해 보시기 바랍니다.

# remountrw                                 # 읽고 쓰기로 전환
# sed -i /dop/s/^/#/' /etc/mpd.conf.sav     # DoP 사용하지 않음
# reboot                                    # 시스템 다시 시작

Native DSD에서 DoP로 바꾸는 것은 아래 명령어입니다.

# remountrw                                 # 읽고 쓰기로 전환
# sed -i /dop/s/^#*//' /etc/mpd.conf.sav    # DoP 사용함
# reboot                                    # 시스템 다시 시작

5. 기타 튜닝

기존 렌더러와 비슷한 경향으로 튜닝을 하였습니다.
들으시는 취향에 맞을 지는 모르겠습니다.

대부분 기존 렌더러와 비슷하므로 이전 글들을 참조하시기 바랍니다.
참고로 제가 알릭스를 갖고 있지 않아서, 앞으로 업데이트할 수 있을 지는 모르겠습니다.
